# Wazi Deploy 

Content | Description
--- | ---
[InstallAutomation](InstallAutomation/) | This folder provides files for installing Wazi Deploy thru an Ansible playbook.
[Reporting](Reporting/) | This category provides templates for querying IBM Wazi Deploy evidence files and generating detailed reports.
[Schemas](Schemas/) | Wazi Deploy schemas used to configure Yaml validation for the config file, the deployment method file and the manifest file.
[zDeploy](zDeploy/) |  Wazi Deploy deployment configuration framework for both Ansible and Python, that allows maintaining application specific configuration along the core deployment configuration for Wazi Deploy.
